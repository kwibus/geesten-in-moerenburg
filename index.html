Geesten in Moerenburg

Dit is een gps speurtocht door de Moerenburg die je kan lopen via je mobiel.

  • Zorg dat gps aanstaat.
  • Inter verbinding hebt.
  • De toch duurt ongveer 45min.
  • Zorg dat je baterij vol genoeg is.
  • Blijf op de padden.
  • Volg de aanwijzing die je krijgt.
  • De start is bij cafe zomerlust.
